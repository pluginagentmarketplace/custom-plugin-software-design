# /roadmap

View detailed roadmap for a specific role.

## Description

This command displays a comprehensive, step-by-step roadmap for any of the 69+ developer roles. Each roadmap includes:

- Learning milestones and checkpoints
- Core technologies and tools
- Recommended resources
- Real-world projects
- Timeline and difficulty progression
- Industry best practices

## Usage

```
/roadmap [role-name]
```

## What It Does

1. **Role Selection** - Choose from 69+ roles or search by keyword
2. **Visual Roadmap** - See the complete learning path from beginner to expert
3. **Detailed Steps** - Understand what to learn at each stage
4. **Resource Links** - Access curated learning materials
5. **Project Ideas** - Get hands-on project suggestions
6. **Timeline** - Realistic estimates for completion
