# /learn

Start a structured learning path for any developer role.

## Description

This command guides you through selecting and beginning a learning path for any of the 69+ developer roles covered in the Software Design & Developer Roadmaps plugin. It helps you:

1. Explore available roles and categories
2. Understand prerequisites and skill requirements
3. Estimate learning timelines
4. Begin your structured learning journey

## Usage

```
/learn
```

## What It Does

1. **Role Discovery** - Browse all 69+ roles across 7 categories:
   - Frontend Development (11 roles)
   - Backend Development (12 roles)
   - Mobile Development (5 roles)
   - Data, AI & ML (10 roles)
   - DevOps & Cloud (9 roles)
   - Databases & Management (7 roles)
   - CS Fundamentals & Specialized (15 roles)

2. **Learning Path Setup** - Get personalized guidance based on:
   - Your current experience level
   - Your career goals
   - Available time commitment
   - Learning style preferences

3. **Structured Progression** - Follow organized steps:
   - Beginner fundamentals
   - Intermediate concepts
   - Advanced techniques
   - Expert specialization

## Related Commands

## Tips

- **Estimated Time**: Most paths take 3-6 months with consistent practice
- **Project-Based Learning**: Each role includes real-world projects
- **Flexible Pace**: Adjust the timeline based on your schedule
- **Combination Paths**: Consider learning complementary roles (e.g., Frontend + Backend for fullstack)
