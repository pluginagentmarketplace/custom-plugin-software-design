# /explore

Explore available roles and their roadmaps.

## Description

This command allows you to browse and explore all 69+ developer roles without committing to a learning path. Perfect for:

- Discovering career paths
- Understanding what each role entails
- Comparing multiple roles
- Planning your development journey

## Usage

```
/explore
```

## What It Does

1. **Category Browser** - Navigate through 7 main categories:
   - Frontend & Web Development
   - Backend & Server Architecture
   - Mobile Development
   - Data, AI & ML
   - DevOps & Cloud Infrastructure
   - Databases & Data Management
   - CS Fundamentals & Specialized Domains

2. **Role Explorer** - For each role, view:
   - Role description and responsibilities
   - Core technologies
   - Typical career progression
   - Average salary and job market demand
   - Real-world applications

3. **Comparison Tools** - Compare roles by:
   - Difficulty level
   - Time to proficiency
   - Technology overlap
   - Career prospects
   - Work environment
